# SportNest Server

SportNest Server is the backend service of the SportNest sports facility booking platform. It manages authentication, database operations, booking systems, facility management, and secure API handling.

The server provides RESTful APIs that connect the client application with MongoDB while maintaining secure and efficient data processing.

The server is designed to support scalable and efficient sports reservation management.

## Installation and Setup

### Clone Repository

```bash
git clone https://github.com/amimahedihasan/sportnest-server.git
```

### Navigate to Server Folder

```bash
cd sportnest-server
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```
